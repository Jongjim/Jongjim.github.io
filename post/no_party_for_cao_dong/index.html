但也没那么心酸

2021-10-30 3 min read

将近下午五点，在同舍友玩完一局飞行棋后打开电脑，收到女友发来的消息，是一张截图，蓝底白字，写着「草東沒有派對鼓手驚傳『陳屍防疫旅館』，警消證實：已死亡多時」，一时有如石沉心海，喘不过气，说不出话。

初听草东没有派对是在二〇一六年，此时的我与独立音乐是井水不犯河水的关系，就像今天的独立「乐迷」与流行「乐迷」们一样，在「你们太商业」与「你们太装屄」这两套叙述中彼此挪揄。至于为什么听草东没有派对，一是抱着「知己知彼，百战不殆」的心理，总得听过方能批评——这一点倒是胜过今天的不少听众；二是那一年周杰伦、五月天、林宥嘉、田馥甄都发行了新专辑，而在知乎的「你认为谁最能夺得金曲奖的某某奖？」问题上，底下异口同声地提到这支不知名的乐队——准备来说，对当时的我来说任何乐队都是不知名的，因为我只知道五月天以及 Beyond。

在网易云音乐播放第一首歌，名为《烂泥》。这不是许志安的歌吗？难道也是一首伤春悲秋的歌曲？点开一听，最先被歌词吸引住：

我想要說的 / 前人們都說過了 / 我想要做的 / 有錢人都做過了

这样的生活有什么意思？不断地消亡不断地降生，不断地踏入不断地走出。——如果在一部长篇小说里，我此时该识破生活的慌言，可在未经启智的头脑中，类似的想法尚未成型便被正确消灭。——所以我只是把《烂泥》添加进歌单，等待下一次消灭。

再之后，他们获得了金曲奖最佳乐团奖，在华晨宇的翻唱下常年活跃在舍友们「他明白 / 他明白」的高呼中。

如何形容草东没有派对对我的影响？与新裤子、后海大鲨鱼一样，他们也是带我走进独立音乐、放下偏见滤镜的人。在《我与李志》中，我提到，复读时使用一部过时的手机，宣泄初级的情绪，而这些也充满了草东没有派对的《丑奴儿》。

在草东没有派对夺得最佳乐团奖的那一届金曲奖，竞争这一奖项的有与草东没有派对气质完全不同的五月天。巧合的是，五月天在同年发行的专辑中唱道「以為的日常 / 原來是無常」，而草东没有派对在《丑奴儿》后发行了一首单曲，名为《如常》，里面同样唱道：

空心的城牆 / 慌亂了日常…偷走了夕陽 / 黑白了無常

对逝者总归是惋惜的，在期许她获得解脱的同时又还是不免自私地想到，她才二十六岁，理应拥有无比光明的未来。
